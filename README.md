# Fake News Detector Using Machine Learning

## Overview

Fake News Detector is a Machine Learning and Natural Language Processing (NLP) application that classifies news articles as Real or Fake. The model uses TF-IDF Vectorization and Logistic Regression to analyze text patterns and provide predictions with confidence scores.

## Features

* Detects fake and real news articles
* Interactive Streamlit web interface
* Confidence score for predictions
* NLP-based text preprocessing
* Fast and user-friendly design
